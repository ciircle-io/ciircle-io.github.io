The Experiment

What is the experiment?

The goal of the experiment is to see if a positive message can have the same viral distribution as a negative message.

Why are your doing this experiment?

To see if we can mitigate the ignorance, fear, and hatred that spreads with negative viral messages.

How can I get involved in the experiment?

The easiest way to get involved is to make the vow to yourself. If you forward the invitation, you are also helping the experiment.

For those wanting to get more involved, there is a mailing list, which I will use for weekly updates.

There is a GitHub project site where I will be posting all the data that I collect. You are free to fork and use the data as you see fit. You also may post issues if you see problems with the data.
